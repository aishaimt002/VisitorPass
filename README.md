5. BoxShadowValue Object Type 
# The box shadow value object is taken by the boxShadow style prop. 

# OffsetX
The offset on the x-axis. This can be positive or negative. A positive value indicate right and negative indicates left. 

# OffsetY
The offset on the y-axis. This can be positive or negative. A positive value indicates up and negative indicates down. 

# BlurRadius 
The larger the value the blurrier the shadow is. Only non-negative values are valid. The default is 0. 

# Spread Distrance 
How much larger or smaller the shadow grows or shrinks. A positive value will grow the shadow, a negative value will shrink the shadow.

# color 
The color of the shadow. The default is black.
